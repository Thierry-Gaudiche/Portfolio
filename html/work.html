Web App "Ma ville accessible"

School project

For this project, I realized: Front / design. This app was coded in Ruby. Ma Ville Accessible is a mobile application for linking to report the lack of accessibility of a site in order to notify an authority (association, town hall) This signaling can be done using a photo or simply thanks to geographic coordinates followed by a short description.
